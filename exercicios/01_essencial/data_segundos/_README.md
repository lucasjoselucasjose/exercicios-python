## Objetivo

Este é um exercício introdutório para praticar a leitura de valores e operações matemáticas básicas. 

## Especificação

### Transforme uma data para segundos

- Abra o arquivo main.py. Dentro dele, você encontrará a função resposta.
- Essa função deverá receber os valores inteiros correspondentes ao dia, hora, minutos e segundos e retornar o total de segundos correspondente a data.
- Lembre-se que retornar um valor é diferente de imprimir um valor.
